活動規範

報名相關

  1. 報名前請先閱讀本活動網站所公開之內容，確認無誤後方可報名。
  2. 未滿18歲者，請先行告知家長或監護人並取得其同意後方可報名。
  3. 惡意報名或報名資格不符者，主辦單位得逕行撤銷其資格而不另行通知。

活動參與相關

  1. 參與本活動的過程中，請將貴重物品隨身攜帶並做好妥善保護。
  2. 為確保活動整體安全，主辦單位會不定時提醒參與活動者有關安全之相關宣導，並請參與活動者隨時注意周遭可疑的人事物。
  3. 為顧及場地之環境清潔及場地之寧靜，請所有參與者務必遵守場地所公告之規定，並隨時自律。
  4. 如需移動至下一個場地時，請務必攜伴前往，避免落單。
  5. 請尊重彼此，勿做有違法律、倫理或道德之約束。

攝影相關

  1. 攝影前可與攝影師做充分討論，讓攝影師了解您想要呈現之拍攝風格。
  2. 攝影前請務必做好妥善安全措施，以保障自身安全。
  3. 攝影時請隨時留意取景角度。主辦單位不允許有任何會造成COSER不舒服之畫面出現。
  4. 若您覺得攝影師所取景之角度會讓您感到不舒服，請立即與主辦單位反映。
  5. 攝影活動中，主辦單位會提供基本道具供COSER借用。如需額外道具，請自行詢問場地管理人員。

其他相關

  1. 違法之情事在本活動中是不被允許的。違者除將被主辦單位請出場並撤銷其資格外，若涉及民事或刑事責任者，主辦單位得逕自尋求法律途徑，絕不寬恕。
  2. 其他未盡事宜，待主辦單位做內部討論後修訂並公告之。
